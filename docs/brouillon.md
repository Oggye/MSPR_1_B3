Les données GTFS fournissent les horaires et les dessertes, mais ne permettent pas d’identifier explicitement les trains de nuit. Nous avons donc intégré la base Back-on-Track, recommandée dans le cahier des charges, comme référentiel métier afin de qualifier les dessertes nocturnes à l’échelle européenne.



* Back-on-Track: *

Parmi les tables disponibles, nous avons sélectionné uniquement les vues métiers permettant d’identifier, qualifier et localiser les trains de nuit. Les tables de type GTFS internes au projet Back-on-Track n’ont pas été intégrées afin d’éviter toute redondance avec les données GTFS opérateurs déjà utilisées.
